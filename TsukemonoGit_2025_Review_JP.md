# 2025年振返り：Nostrエコシステムと共に駆け抜けた開発の記録

2025年も残すところあとわずかとなりました。
今年の自分のGitHubアクティビティを振り返ってみると、ひたすらコードを書き、Nostrという新しいインターネットの形を模索し続けた1年だったと感じます。

## 今年の中心：lumilumiの開発
今年、自分の中で最も大きなウェイトを占めていたのは、間違いなく **[lumilumi](https://github.com/TsukemonoGit/lumilumi)** です。
年間で **1,781回** ものコミットを積み上げ、Nostr Webクライアントとしての完成度を高めるために心血を注いできました。単なる機能追加だけでなく、ユーザー体験をどう磨き上げるか、試行錯誤の連続でした。

## 広がり続けるNostrの世界
lumilumi以外にも、今年は多くのプロジェクトに挑戦しました。
- **自動化とBot:** **[nostr-monoGazo-bot](https://github.com/TsukemonoGit/nostr-monoGazo-bot)** や **[nostr-iroiro-bot](https://github.com/TsukemonoGit/nostr-iroiro-bot)** を通じて、Nostr上での新しいインタラクションの形を探りました。
- **開発者向けツール:** **[nostr-web-components](https://github.com/TsukemonoGit/nostr-web-components)** や **[nostr-content-parser](https://github.com/TsukemonoGit/nostr-content-parser)** など、他の開発者が使える「部品」作りにも力を入れました。
- **表現の場:** **[longFormEditor](https://github.com/TsukemonoGit/longFormEditor)** (makimono) では、短文だけではない「長文」という表現スタイルにもフォーカスしました。

## 開発のリズム
振り返ってみると、特に **2025年の夏** は自分でも驚くほどの集中力でした。
特に **7月（801件）** と **8月（664件）** は、創作意欲が爆発していた時期だったと思います。年末にかけても、その勢いを止めることなく走り抜けることができました。

## 数字で見る2025年
- **総コントリビューション数:** 4,720件
- **1日平均:** 約13件
- **主な使用技術:** Svelte, TypeScript, Python

## 2026年に向けて
2025年は、一貫して「作り続けること」を大切にした1年でした。Nostrというプロトコル、そしてSvelteという技術を使って、自分なりのインターネットの形を少しずつ形にできた気がします。

来年もこの歩みを止めることなく、さらに面白いものを作っていきたいと思います！
